## 什么是 VCLight ？

VClight 是一个适用于 Vercel 的 Serverless 函数框架。它可以帮助你在一个云函数中处理所有页面，以建立一个完整的站点。 

你可以用它建立一个动态站点，以弥补 Vercel 只能建立静态站点的不足。

VCLight 的基于插件的。你可以编写插件并应用，也可以使用第三方模块。
